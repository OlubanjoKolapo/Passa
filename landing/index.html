Agents can decide, but they can't pay.

Every agent-payment tool assumes a card — most of the world doesn't use one.

India is solving it for UPI. Visa for cards.

Everyone's solving one rail at a time. Passa connects them all into a single endpoint.

The orchestration layer.

Passa decides whether a payment is allowed, routes it, executes it, and handles fallbacks.

Start building in minutes

1

Install the SDK

Terminal
$ npm install @passa/sdk
2

Configure

agent.js
import Passa from '@passa/sdk';
const passa = new Passa(process.env.PASSA_KEY);
3

Execute

payment.js
await passa.pay({ 
  amount: 2300, 
  currency: 'USD', 
  merchant: 'ifood' 
});
Passa is in sandbox. No real funds move.
